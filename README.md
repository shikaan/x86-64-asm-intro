# x86-64-asm-intro
Learning x86-64 assembly with nasm

## Tools

### nasm

```sh
sudo apt install nasm
```

### shmux

```sh
sudo sh -c "wget -q https://shikaan.github.io/sup/install -O- | REPO=shikaan/shmux sh -"
```

## Execute

```sh
shmux build -- file.asm

./file.asm.out
```
